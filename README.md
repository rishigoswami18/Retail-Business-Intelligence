# Retail Business Intelligence & Customer Analytics

## 📊 Project Overview
This project provides a comprehensive end-to-end analysis of a Retail Supply Chain dataset ($2.3M+ in sales). The goal was to identify revenue leakages, optimize discounting strategies, and segment customers for targeted marketing using **Python, SQL, and Streamlit**.

## 🚀 Live Insights Summary
- **Total Revenue**: $2,297,200.86
- **Total Profit**: $286,397.02
- **Revenue Leakage**: Discounts above 20% lead to negative returns.
- **Top Region**: West ($108K profit).

## 📂 Repository Structure
- `scripts/`: Python automation for Data Cleaning, EDA, and RFM Segmentation.
- `sql/`: Advanced analytical queries (YoY Growth, Ranking, etc.).
- `dashboard/`: Interactive Streamlit Web Application.
- `data/`: Raw and cleaned datasets (Git ignored by default).
- `visualizations/`: Automated reports generated by the EDA pipeline.

## 🛠️ Tech Stack
- **Language**: Python (Pandas, Plotly, Seaborn)
- **Database**: SQL (Window Functions, CTEs)
- **BI Tool**: Streamlit (Interactive UI)
- **Methodology**: RFM Analysis, Time-Series Forecasting

## ⚡ Quick Start
1. **Clone the repo**:
   ```bash
   git clone https://github.com/yourusername/Retail_BI_Project.git
   ```
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the Data Pipeline**:
   ```bash
   python scripts/data_cleaning.py
   python scripts/eda_analysis.py
   python scripts/customer_segmentation.py
   ```
4. **Launch the Dashboard**:
   ```bash
   streamlit run dashboard/app.py
   ```
